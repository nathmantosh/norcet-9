# Security Policy

## Reporting a Vulnerability

If you find a security issue, please open a private report through GitHub's security reporting flow if it is available for this repository. If private reporting is not available, open an issue with a minimal description and avoid posting sensitive exploit details publicly.

This is an educational frontend app and should not store secrets in source control. Do not commit API keys, tokens, private student data, or exam materials that cannot be shared publicly.

## Supported Versions

The `main` branch is the active development version.
